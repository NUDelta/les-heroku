Notification Preferences

In this section, please tell us if you would like to be notified by LES if it knows certain information about a Workspace nearby. These responses will be used by LES to determine when to notify you about Workspaces.

If you respond No to a prompt below, you will not be notified by LES when it knows that specific piece of information about a nearby Workspace. However, you will still be notified about prompts that you respond Yes to even if they are below one you said No to.

For example, if you respond:
LES knows that a nearby workspace has private seating (individual tables) available. No
LES knows that a nearby workspace has private seating (individual tables) near outlets available. Yes
then LES would not notify you about private seating (individual tables) generally but would notify you about private seating (individual tables) near outlets.
